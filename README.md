## Create referral Enpoint

**Endpoint:**

```
localhost:8080/v1/referral
```

**Request Body:**

```json
{
  "referrerName": "abc",
  "referrerEmail": "temp@mail.com",
  "referrerPhone": "51515155155",
  "refereeName": "jk",
  "refereeEmail": "temp@mail.com",
  "refereePhone": "5555555555",
  "course": "data-science"
}
```
